# OWS [Optician Workshop System]

## Requirements 
- PHP 8.1+ 
- Composer 2.2.6 + 
- MYSQL or MariaDB 
- ### *Laravel 10 Server Requirements* : 
    - PHP >= 8.1
    - Ctype PHP Extension
    - cURL PHP Extension
    - DOM PHP Extension
    - Fileinfo PHP Extension
    - Filter PHP Extension
    - Hash PHP Extension
    - Mbstring PHP Extension
    - OpenSSL PHP Extension
    - PCRE PHP Extension
    - PDO PHP Extension
    - Session PHP Extension
    - Tokenizer PHP Extension
    - XML PHP Extension
